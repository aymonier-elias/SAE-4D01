## 🎯 Contexte
Les étudiants de GEII développent un système de ruches connectées collectant différentes métriques telles que le poids, la température, l’humidité ou les données GPS. Ils souhaitent que les étudiants de MMI créent une interface permettant :  

- la visualisation des données sous forme de graphiques (par exemple sur l’année écoulée),  
- la consultation personnalisée pour chaque apiculteur enregistré via un système d’authentification et d’administration,  
- l’enregistrement de données supplémentaires pour le suivi des ruches.  

L’interface doit être ergonomique et esthétique, avec une page d’accueil présentant clairement le projet. Il faudra également prendre en compte que la communauté des apiculteurs est majoritairement âgée, et adapter le design en conséquence.
